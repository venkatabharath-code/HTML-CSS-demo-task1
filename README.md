# SnapX Photography Website

SnapX Photography is a modern and fully responsive photography contest website built using pure HTML and CSS without using Bootstrap or JavaScript. The project is designed to replicate a professional photography competition platform with clean UI, responsive layouts, smooth spacing, and visually attractive sections.

The website includes a modern navigation bar, fullscreen hero section, featured photography gallery, contest categories, winners showcase section, pricing plans, and footer. Special attention was given to responsiveness so the website works properly on desktop, tablet, and mobile devices.

This project helped in improving frontend development skills such as Flexbox, CSS Grid, responsive design, spacing management, hover effects, layout structuring, and modern UI styling techniques.

## Features

* Fully Responsive Design
* Pure HTML5 and CSS3
* Modern Photography UI
* Mobile Friendly Layout
* Contest Categories Section
* Winners Showcase
* Pricing Plans Section
* Clean and Organized Code
* No Bootstrap
* No JavaScript

## Technologies Used

* HTML5
* CSS3
